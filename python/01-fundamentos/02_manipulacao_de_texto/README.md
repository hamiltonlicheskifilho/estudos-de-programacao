## Manipulação de Texto em Python

## Objetivo
Aprender a trabalhar com strings (textos) de forma eficiente em Python.

## Exercícios Realizados
1. **Analisador de Textos** 
2. **Primeira e Última Ocorrência de uma String** 
3. **Primeiro e Ultimo Nome de uma Pessoa** 
4. **Procurando uma String Dentro de Outra** 
5. **Separando Digitos de um Número** 
6. **Verificando as Primeiras Letras de um Texto.** 

## Conceitos aplicados:
- **Concatenação** com `+`
- **Métodos** `.upper()`, `.lower()`, `.title()`
- **`.strip()`** - Remove espaços extras
- **`.split()`** - Divide string em lista
- **`.find()`** - Busca em texto
- **`.count()`** - Contar o número de ocorrências
- **`.upper()`** - Converter caracteres minúsculos em caracteres maiúsculos. 
- **`.zfill()`** - Preencher uma string com zeros
- **Acesso por índice** `[0]`, `[-1]`
- **Intervalos** `[inicio:fim]`
- **Passos** `[inicio:fim:passo]`
