# Cálculos Básicos em Python

## Objetivo
Dominar as operações matemáticas fundamentais e a sintaxe básica do Python.

## Exercícios Realizados
1. **Aluguel de Carros** 
2. **Calculando Desconto** 
3. **Conversor de Medidas** 
4. **Conversor de Moedas** 
5. **Media Aritmética** 
6. **Pintando Parede** 
6. **Reajuste Salarial** 

## Conceitos aplicados:
### 1. Operações Aritméticas
- **Soma, subtração, multiplicação e divisão**
- **Divisão** (`/`) e **módulo** (`%`)

### 2. Formatação de Resultados
- **Limitação de casas decimais** com `:.2f`
