# Condições Básicas - if/else

## Objetivo
Aprender a tomar decisões no código usando estruturas condicionais simples.

## Exercícios
1. **Jogo da Adivinhação**
2. **Radar Eletrônico**
3. **Par ou Ímpar**
4. **Custo da Viagem**
5. **Ano Bissexto**

## Conceitos aplicados:
- Estrutura `if` / `else`
- Operadores de comparação (`>`, `<`, `==`)
- Operador módulo (`%`) para par/ímpar`
- Condições compostas com `and`/`or`
