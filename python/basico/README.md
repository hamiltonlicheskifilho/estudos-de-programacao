# Básico

Esta pasta contém exercícios iniciais de Python, focados nos fundamentos da linguagem e no desenvolvimento da lógica de programação.

Os exercícios abordam temas como entrada e saída de dados, tipos primitivos, operações matemáticas, formatação de saída e resolução de pequenos problemas práticos.

O objetivo é praticar conceitos básicos de forma progressiva, servindo como base para conteúdos mais avançados.
