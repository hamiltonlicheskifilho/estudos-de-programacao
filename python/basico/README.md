# Estudos de Programação em Python

Este repositório contém exercícios de fundamentos em Python, organizados por tema e nível de dificuldade.  
O objetivo é treinar conceitos básicos e intermediários, incluindo cálculos, manipulação de texto, uso de módulos e listas.

---

## Estrutura do Repositório

- `01_calculos_basicos.py`  
  Exercícios de cálculos matemáticos simples: média, conversões, operações básicas.  

- `02_manipulacao_de_texto.py`  
  Exercícios envolvendo strings, análise de texto, separação de dígitos e manipulação de nomes.  

- `03_uso_de_modulos.py`  
  Exercícios utilizando módulos do Python (`math`, `time`, etc.) para cálculos e operações.  

- `04_listas_e_sorteios.py`  
  Exercícios com listas, sorteios e manipulação de itens de forma simples.

---

## Aprendizado e Tecnologias

- Python 3.x

- Operações matemáticas básicas

- Strings e manipulação de texto

- Uso de módulos (math, random, time, pygame em casos especiais)

- Estruturação de scripts e organização de repositório
