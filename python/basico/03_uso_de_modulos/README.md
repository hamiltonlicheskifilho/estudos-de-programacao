# Uso de Módulos em Python

## Objetivo
Aprender a expandir as funcionalidades do Python através de módulos e bibliotecas.


## Exercícios
1. **Catetos e Hipotenusa**
2. **Quebrando um Número**
3. **Seno, Cosseno e Tangente**

## Conceitos aplicados:
### . Módulos Matemáticos
- **`math`** - Funções matemáticas avançadas
- **`sin, cos, tan, radians`** - Seno, Cosseno e Tangente
- **`trunc`** - Quebrando um número
- **`hypot`** - Catetos e Hipotenusa
